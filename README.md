# 📝 Text Summarizer GUI using Python & NLTK

This is a simple Python GUI application that summarizes large text using Natural Language Processing (NLP) with the **NLTK** library and **Tkinter** for the user interface.

---

## 🚀 Features

- 📄 Accepts long input text
- 🧠 Summarizes it into top 3 sentences
- 🖥️ Easy-to-use Tkinter-based GUI
- 🧹 Clear input/output fields with a button click
- 🔡 Uses tokenization and stopword filtering for better summarization

---

## 📦 Requirements

- Python 3.x
- NLTK
- Tkinter (included with Python)
